Topineur Documentation
======================

Topineur is an object-functional contractual paradigm (OF-C) designed to blend
the reasoning strengths of pure functional programming with the ergonomics of
object-oriented modelling.  This documentation mirrors the structure commonly
used by the Python language reference to ease discovery and long-term
maintenance.
